# Trabalho Final - Integração Contínua de Automação de Testes

## Objetivo

Este projeto foi desenvolvido para a disciplina Integração Contínua da Pós-Graduação em Automação de Testes do Júlio de Lima.

## Tecnologias

- Node.js
- Playwright
- GitHub Actions
- Yarn

## Workflows

- Execução Manual
- Execução por Push
- Execução Agendada
- Execução após outra pipeline

## Como executar

yarn install

npx playwright install

yarn run e2e

## Relatórios

Os relatórios do Playwright são publicados como Artifacts do GitHub Actions.
